DESCRIPCION:

Este código implementa un juego completo de Tres en Raya (Tic-Tac-Toe) en Python con Pygame, donde el humano juega como X (MAX) y un agente de ia como O (MIN) 
usando el algoritmo Minimax. La interfaz muestra el tablero principal a la izquierda y un árbol de decisiones visual interactivo a la derecha, 
ideal para entender cómo piensa la IA

INSTALACIÓN Y USO:
_Instala Pygame: pip install pygame
_Ejecuta: python game.py
